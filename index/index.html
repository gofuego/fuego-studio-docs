Host private sites. Edit docs in place.

fuego-studio is an authenticating host for GitHub-backed static sites, with in-place editing for Fuego sites.

It serves a repo's built output behind GitHub-permission gating — GitHub Pages for private sites — and, for people who already have write access, lets them edit a page's source right on the page and commit it straight back to GitHub. No CMS, no database, no separate preview app: the gate, the site, and the editor are one origin.
